# FUTURE_CS_01 — Vulnerability Assessment Report

## Task 1: Vulnerability Assessment for a Live Website

### Target
- Website: http://zero.webappsecurity.com

### Tools Used
- Nmap 7.99 — Port scanning and service detection
- OWASP ZAP (Passive) — Vulnerability scanning
- Browser DevTools — HTTP header and cookie analysis

### Key Findings
- 3 HIGH risk vulnerabilities
- 4 MEDIUM risk vulnerabilities
- 1 LOW risk vulnerability
- 2 Informational findings
- Total: 10 vulnerabilities identified

### Files in this Repository
- Vulnerability_Assessment_Report.docx — Full detailed report
- Vulnerability_Assessment_Presentation.pdf — Canva presentation
- Screenshots/ — Evidence from all tools used

### Summary
A full vulnerability assessment was conducted using passive
scanning techniques on Zero Bank web application. All findings
include risk classification, business impact explanation,
and remediation steps.
